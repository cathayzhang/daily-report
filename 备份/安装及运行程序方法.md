# 开发日报工具 - 安装及运行指南 (小白专用版)

欢迎使用！本指南将手把手教您如何在任何一台新电脑上，从零开始安装、配置并成功运行开发日报工具。

请放心，整个过程就像安装普通软件一样简单，您只需要跟着步骤做，不需要任何编程知识。

---

### **第一步：把"工具文件夹"带到新电脑**

这个工具的所有文件都在一个名为 `开发日报` 的文件夹里。您需要把它从家里电脑带到公司电脑。

**推荐方法：**
使用一个U盘，将整个 `开发日报` 文件夹从您的家用电脑上完整地复制过来。

> **注意**：如果您的同事帮您设置过，文件夹里可能有一个叫 `venv` 的子文件夹。在复制前，最好**将它删除**，因为它只对您之前的电脑有效。

---

### **第二步：安装两个"必备软件"**

我们的工具需要两个"发动机"才能运转起来。如果您的电脑上没有，需要先安装一下。

#### **1. 安装 Python (工具的"主发动机")**

Python 是这个工具运行的基础。

1.  **下载**：打开浏览器，访问 Python 官网下载页面：[https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/)
    *   点击页面顶部最新的稳定版"Latest Python 3 Release"链接。
    *   在新页面中，拉到最下方，找到 `Windows installer (64-bit)` 并点击下载。

2.  **安装**：双击下载好的 `.exe` 文件。
    *   **【极其重要！】** 在安装界面的最下方，**务必勾选 "Add Python.exe to PATH"** 这个选项！
    *   然后点击 "Install Now" 即可。程序会自动完成安装。

#### **2. 安装 Git (一个"下载工具")**

Git 能帮助我们更方便地管理代码。

1.  **下载**：打开浏览器，访问 Git 官网下载页面：[https://git-scm.com/download/win](https://git-scm.com/download/win)
    *   页面会自动开始下载适用于您系统的安装包。

2.  **安装**：双击下载好的文件。您不需要做任何修改，一路点击 **"Next"**，最后点击 **"Install"** 即可。

---

### **第三步：让工具"准备就绪"**

现在，我们需要让工具和您的新电脑"磨合"一下。

1.  **打开"命令提示符" (CMD)**
    *   点击电脑左下角的"开始"菜单。
    *   直接用键盘输入 `CMD`，然后按下回车键。
    *   一个黑色或蓝色的窗口会弹出来，这就是我们接下来要用的"命令提示符"。

2.  **进入"开发日报"文件夹**
    *   您需要告诉电脑，我们要操作哪个文件夹。请复制您 `开发日报` 文件夹的完整路径。
    *   在命令提示符里，输入 `cd ` (注意 cd 后面有一个空格)，然后粘贴您刚才复制的路径，最后按回车。
    *   例如：`cd C:\Users\YourName\Desktop\开发日报`

3.  **创建"隔离工作区"**
    *   为了不影响您电脑的其他文件，我们为工具创建一个独立空间。在命令提示符中输入以下命令并按回车：
        ```
        python -m venv venv
        ```
    *   这会在您的文件夹里新建一个叫 `venv` 的子文件夹。

4.  **进入"隔离工作区"**
    *   继续在命令提示符中输入以下命令并按回车：
        ```
        venv\Scripts\activate
        ```
    *   成功后，您会看到命令行路径的最前面出现了 `(venv)` 字样。

5.  **安装所有"零件"**
    *   最后一步，工具需要安装一些必要的"零件"才能运行。输入以下命令并按回车：
        ```
        pip install -r requirements.txt
        ```
    *   电脑会自动从网上下载并安装所有需要的零件。这个过程可能需要几分钟，请耐心等待它跑完。

---

### **第四步：开始使用！生成您的第一份报告**

所有准备工作都已完成！现在您可以随时生成报告了。

1.  **确认您的数据文件**
    *   将您从Jira导出的数据文件（比如 `jira_data.csv`）放入 `data` 文件夹中。

2.  **运行程序**
    *   确保您的命令提示符窗口还开着，并且路径前面有 `(venv)` 字样。
    *   输入以下命令，然后按回车：
        ```
        python main.py data/你的数据文件名.csv
        ```
    *   **注意！** 请将 `你的数据文件名.csv` 替换为您真实的文件名！

3.  **查看报告**
    *   当程序运行完毕并提示"恭喜！报告已成功生成"时，您就可以去 `output/reports` 文件夹里查看了。
    *   一个以当天日期命名的 `.html` 文件就是您的报告。用您的浏览器（如Chrome, Edge）打开它即可。

希望这份指南能帮助您顺利完成安装！如果遇到任何问题，可以带着这份文档向您的技术同事求助。
